# Data Layout

This folder hosts the *local* dataset and derived artifacts. It is intentionally not versioned in full.

## Structure
- `raw/` — Preprocessed JSONs (treated as raw data input).
- `features/` — Extracted graph features (eigenvalues, edge counts).
- `splits/` — Train/validation/test splits and normalized test split.
- `cache/` — Dataset cache files for faster training.
- `meta/` — Metadata and configuration files.

## Expected Metadata
- `meta/programs_metadata.txt` — Line‑paired index → program name mapping.
- `meta/clone_search_config.json` — Clone search config generated by scripts.
- `meta/split_stats.txt` — Split statistics summary.

## How to Regenerate Metadata
```bash
python scripts/meta_generation/build_clone_search_config.py
python scripts/meta_generation/summarize_projects.py
```
