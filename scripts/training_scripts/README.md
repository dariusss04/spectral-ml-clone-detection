# Training Scripts

All training scripts write **metrics.csv**, **summary.csv**, and **weights.pth**
into the corresponding folder inside `experiments/`.

## Scripts
- `train_mlp_activation_sweep.py` → `experiments/mlp_activation_sweep/`
- `train_mlp_depth_sweep.py` → `experiments/mlp_depth_sweep/`
- `train_deepset_full.py` → `experiments/deepset_full/`
- `train_deepset_eigenonly.py` → `experiments/deepset_eigen_only/`
- `train_self_attention_full.py` → `experiments/self_attention_full/`
- `train_self_attention_eigenonly.py` → `experiments/self_attention_eigen_only/`
